# AI DevOps Demo API

Simple Node.js Express API for demonstrating the AI DevOps Platform capabilities.

## Features

- ✅ Express REST API
- ✅ Health check endpoint
- ✅ Jest unit tests with 100% coverage
- ✅ CI/CD pipeline ready
- ✅ Docker-ready
- ✅ AWS ECS deployment configuration

## Quick Start

### Install Dependencies
```bash
npm install
```

### Run Locally
```bash
npm start
```

Server runs on: `http://localhost:3000`

### Run Tests
```bash
npm test
```

## API Endpoints

### `GET /`
Welcome message and API status

### `GET /health`
Health check with uptime information

### `GET /api/hello?name=YourName`
Personalized greeting endpoint

### `POST /api/echo`
Echo back the request body
